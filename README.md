# 🩸 Blood Donation Website

A modern blood donation website built to connect donors and receivers easily.  
This project helps users learn about blood donation, register as donors, and access important information through a clean and responsive interface.

## 🚀 Features
- Responsive UI
- Donor information section
- Blood donation awareness
- Clean design
- Fast and lightweight website

## 🛠️ Built With
- HTML
- CSS
- JavaScript
