# Blood Donation Website

A local browser-based blood donation directory for saving donor records, checking blood type availability, and viewing demo hospital inventory for Belagavi.

## Features

- Register with a name and phone number.
- Save donor records locally in the browser.
- Search saved donors by blood type.
- View demo hospital blood inventory.
- Delete only the donor records registered with the signed-in phone number.

## Run Locally

Open `index.html` in a browser.

No build step or package installation is required.
